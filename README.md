# smol
Small programming language.

## Getting started
1. [Download](https://rust-lang.org/learn/get-started/) the Rust toolchain.
2. Clone the repo: `git clone https://github.com/lauchimoon/smol`
3. Build the project in release mode: `cargo build --release`
4. Run `./target/release/smol <smol program>`

Check [the examples directory](examples/) to see some `smol` programs.
